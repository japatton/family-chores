# Code of Conduct

Family Chores is a personal open-source project, and interactions around it — issues, pull requests, discussions, review comments — should stay friendly and focused on the work. To make the ground rules explicit, this project adopts the [Contributor Covenant, version 2.1][covenant], without modification, as its contributor code of conduct.

The full text — expected behaviour, unacceptable behaviour, enforcement guidelines — lives at the link above. If you haven't read the Covenant before, it's short and worth a skim.

## Reporting

Report violations to **`YOUR_CONTACT_EMAIL_HERE`**. Reports are read by the project maintainer only. Expect acknowledgement within 5 business days.

If the behaviour you're reporting also involves a security or privacy issue (e.g. someone leaked data or credentials through an issue or PR), use the [security channel](SECURITY.md) instead — it's private by default on GitHub.

If the complaint is about the maintainer, escalate via GitHub's [abuse-reporting flow][gh-abuse] — I can't impartially adjudicate a report about myself.

## Enforcement

Consequences follow the Community Impact Guidelines described in the Covenant, which range from a private clarifying note to a permanent ban depending on severity and pattern. Enforcement actions are documented privately; public callouts are reserved for the clearest cases.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
[gh-abuse]: https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam
